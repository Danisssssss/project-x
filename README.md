Установка
1) Клонирование репозитория
```bash
git clone git@github.com:Danisssssss/project-x.git
```
2) Установка зависимостей
```bash
node -v
npm -v
```
```bash
npm install
```
3) Запуск проекта
```bash
npm run dev
```
